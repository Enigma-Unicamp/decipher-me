Your browser does not support the canvas element.

- Instruções:
-- 'movl N' - move ambos os robôs N quadrados para a esquerda, onde N é um número entre 1 e 9.
-- 'movr N' - move ambos os robôs N quadrados para a direita, onde N é um número entre 1 e 9.
-- 'movu N' - move ambos os robôs N quadrados para cima, onde N é um número entre 1 e 9.
-- 'movd N' - move ambos os robôs N quadrados para baixo, onde N é um número entre 1 e 9.
-- 'swi' - se existe um bloco ON/OFF adjacente a um dos robos, ele é (des)ativado.
-- * Se ambos os robôs estão próximos a blocos ON/OFF, nada acontece.
